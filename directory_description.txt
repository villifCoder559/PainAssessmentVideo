
## Directory Descriptions

### `custom/`
Contains custom modules and scripts for this project.

### `extract_feature.py`
Script for extracting features from the video dataset.

### `history_run/`
Contains directories for different runs of the model training and evaluation.

### `local_model_directory/`
Contains local model configurations and preprocessor settings.
- `preprocessor_config.json`: Configuration file for the preprocessor.

### `partA/`
Contains the initial data and video files.
- `starting_point/`: Initial CSV files and other starting data.
- `video/video`: Video files used for the project.
- `video/features/`: Contains the extracted features. Each folder includes a README file with details about the contents.

### `requirements.txt`
Lists the dependencies required for the project.

### `test.ipynb`
Jupyter notebook for testing and experimenting with the model and data.

### `VideoMAEv2/`
Contains the VideoMAEv2 model from https://github.com/OpenGVLab/VideoMAEv2
and related scripts.
